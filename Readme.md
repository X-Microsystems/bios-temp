# Temporary BIOS Repository
This repo contains an unfinished XMICRO-6502 BIOS snapshot for review. Note that GitHub's preview does not properly render the 8-character tab stops.

Some points of interest:<br>
[xmcf-cf.s](https://github.com/X-Microsystems/bios-temp/blob/main/xmcf-cf.s) - CompactFlash storage driver<br>
[xmcf-rtc.s](https://github.com/X-Microsystems/bios-temp/blob/main/xmcf-rtc.s) - Real-time clock driver<br>
[xms-ps2.s](https://github.com/X-Microsystems/bios-temp/blob/main/xms-ps2.s) - PS/2 keyboard driver<br>
